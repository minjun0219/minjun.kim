# 김민준

키즈노트, 와디즈 등에서 프론트엔드 개발팀을 리드하며 레거시 시스템을 React/Next.js 기반으로 전환하고, CI/CD 파이프라인 구축부터 디자인 시스템 도입까지 개발 환경 전반을 개선해왔습니다. 10년 된 Django 서비스를 현대적인 프론트엔드 아키텍처로 마이그레이션하며 개발 효율성 60% 향상, 코드 재사용율 70%를 달성한 경험이 있습니다.

## 경력

### 키즈노트 (2020.10 – 2025.10)

- Django 레거시 시스템을 Next.js/React 기반으로 전면 마이그레이션
- Docker 컨테이너화, Jenkins/GitHub Actions CI/CD, Blue/Green 무중단 배포 환경 구축
- Nx 모노레포 기반 웹뷰 서비스 모듈화 및 JS Bridge 라이브러리 NPM 패키지 배포
- Sentry 도입으로 런타임 오류 모니터링 체계 구축

### 와디즈 (2017.04 – 2020.08)

- ActiveX 기반 투자 프로세스를 가상계좌 방식으로 전면 개편

- 레거시 댓글 시스템 Ajax 전환으로 로딩 성능 90% 개선 (4MB → 경량화)
- 디자인팀과 협업하여 사내 디자인 시스템 및 컴포넌트 라이브러리 구축

### 남전사 (2014.07 – 2016.12)

## 사이드 프로젝트

### [ogpeek](https://ogpeek.dev) [GitHub](https://github.com/minjun0219/ogpeek)

- URL 입력만으로 Open Graph 메타데이터를 파싱하고 OGP 명세 위반 사항(제목 길이, 이미지 누락, URL 불일치 등 12가지 규칙)을 자동 검증해주는 도구
- 순수 JavaScript 엔진 패키지(`packages/ogpeek`)와 Next.js 15 기반 웹사이트(`website`)를 모노레포로 구성, Cloudflare Workers(OpenNext)로 배포

### [hotwatermat-ble](https://github.com/minjun0219/hotwatermat-ble)

- 온수매트 앱의 불편함을 해소하고자 AI를 활용해 블루투스 통신을 리버스 엔지니어링하여, 바이브 코딩으로 CLI를 개발해 AI 에이전트로 제어할 수 있게 만든 프로젝트
